Merge data từ file Name.csv và file Point.csv để xuất ra file output Rank.csv gồm các field:
- Student ID
- Name
- Average: Được tính bằng (Math + Geography + Chemistry) / 3
- Rank: Được đánh giá theo Average
	0 =< Average < 4 (F)	
	4 =< Average =< 5 (D)	
	5 < Average =< 6 (C)
	6 < Average =< 8 (B)
	8 < Average =< 10 (A)
